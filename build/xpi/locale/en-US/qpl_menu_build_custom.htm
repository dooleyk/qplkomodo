Build Custom Files

This dialog lets you run the QPL compiler using any of its command line arguments. While the QPL Build dialogs let you easily select commonly used options, here you can use other options to create files for special purposes. For example, you can use the -rpt option to build a file that contains SQL snippets for each question that can be used to create SQL statements that can be used to label the coded survey responses.

Options

Type your options into the "Enter custom options" field, and then click the Build button to build your files. Check the Command Output pane to what files were created. These files will not be automatially opened in Komodo.

Enter custom options: [-o file] filename[.qpl] [modifying option...]
              [program option...] [report option...]


Redirect Console Option
  -o file       Redirect console output from screen to this file
                (Must be first option.)

Program Options 
  -htm          HTML files for local testing or emailing
  -php          PHP files for server deployment
  -sas [# [#]]  SAS system file
  -spc [# [#]]  SPSS/PC+ system file
  -sps [# [#]]  SPSS for Windows system file
  -spx [# [#]]  SPSS-x system file

  Note: Optional numbers for SAS and SPSS change the maximum
        value and variable label lengths used.

Report Options 
  -x1   List Name
  -x2   List Name, Alias, Sequence, Type, Length
  -x3   List Name, Alias, Sequence, Type, Length, Hidden, Page, Layout, Text
  -txt  Simple version of questionnaire in text format
  -rpt [var...] JavaScript file of query-builder arrays
