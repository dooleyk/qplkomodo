Popup Text Box

This dialog box may be used to create the QPL code for one "pop-up" information box. This code will be inserted into your program at the current cursor position.

Popup boxes usually contain extra information about a question. You can create a hypertext link to a popup information box that you have created by inserting its name delimited by caret (^) characters into the text of your question. When you view the questionnaire page, this will be replaced with a hypertext link using the phrase you entered for the popup's title.

You may use any number of popup boxes, and may insert any number of links to a popup box in the text of of question.

Name The name of a popup must start with a letter, contain only letters and numbers, and be from one to sixteen characters long.

If a name is not specified, then a name will be created automatically based on its order in your program. The first one will be named POPUP1, the second will be POPUP2, etc.
Title The title entered here will be used to label the popup box when it is displayed in a separate browser window, and the hypertext link when a popup name is put into the text of a question.
Width Set the width of the popup browser window in pixels (default is 300 pixels).
Height Set the height of the popup browser window in pixels (default is 300 pixels).
Contents Enter the text that will be put into the popup box. (See also Text Line Formatting.)
Use print button Check this option to include a print button on the popup browser window (default).
Use close button Check this option to include a button that will close the popup browser window (default).
Hover text over link Check this option to show popup text when the respondent hovers their mouse pointer over the popup link instead of a separate text box window.

See also Text Line Formatting.
