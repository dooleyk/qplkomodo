Product Block

This dialog box may be used to create the QPL code for needed to create a GAO E-supplement product or a technical appendix to be added to a printed report product. This code will be inserted into your program at the current cursor position.

Product type e-Supplement - Insert commands for e-supplement product. (Also complete the fields in the E-Supplement group below.)

Technical Appendix - Insert commands for technical appendix for printed report. (Also complete the fields in the Technical Appendix group below.)
Primary Product
Title Enter the name of the primary product (Ex. Embassy Construction: Additional Actions Are Needed to Address Contractor Participation).
Number Enter the GAO product number (Ex. GAO-15-48).
Issue date Enter the date the product will be issued (Ex. January 16, 2015).
E-Supplement
Title Enter the name of the e-supplement product. (Ex. New Embassy Construction: Structured Interviews with Construction Firms)
Number Enter the GAO product number (Ex. GAO-09-47SP)
Issue date Enter the date the product will be issued (Ex. January 2009)
Job code Enter the job code for the work on the e-supplement (Ex. 320633).
Contact Enter the GAO contact information for the e-supplement (Ex. If you have questions concerning these data, please contact Jess Ford at 202-512-4128 or Terrell Dorn at 202-512-6923.).
Background Enter the Backgound section information that will appear on the home page for this e-supplement.

Example

This document presents information from a series of structured interviews GAO held with 17 of the 21 U.S. construction firms that won contracts to build new U.S. embassies, consulates, and other diplomatic facilities. Our structured interviews were designed to assess contractor views and experiences on a wide range of construction-related topics, including, (1) experiences working with the Department of State (State), (2) StateÂ’s program-level and onsite construction management policies and processes, (3) incentives for pursuing construction awards, (4) challenges in completing NEC and related construction projects, and (5) profitability of NEC and related projects. We conducted the interviews from May 2008 to June 2008 and all of our work in accordance with generally accepted government auditing standards. For a detailed discussion of our scope and methodology, see Embassy Construction: Additional Actions are Needed to Address Contractor Participation GAO-09-48
Technical Appendix
Title Enter the title of this appendix. (Ex. Appendix III: Structured Interviews with Construction Firms)
Issue date Enter the date the product will be issued (ex. January 2009)
Job code Enter the job code for the work on the primary product (Ex. 320632).
Background Enter the Backgound section information that will appear at the top of the first appendix page.
Statistical Options
BY variable Enter the name of a question to be used as a SAS BY variable in the statistical analysis program.
CLASS variable Enter the name of a question to be used as a SAS CLASS variable in the statistical analysis program.
WEIGHT variable Enter the name of a question to be used as a SAS WEIGHT variable in the statistical analysis program.
System Select source of SAS system file that will be used to created the report.htm file.
Sudaan Estimate This setting is only used if SAS system file was created by Sudaan. Option indicates the type of estimate that was made using Sudaan.
