This Komodo Edit/IDE Add-on provides a GUI interface to running the QPL 
command line programs. 

After installing, update your the QPL6 Preferences (Options in Windows)
to select your computer's operating system. The QPL binaries are included
for Win32, Linux32, Linux64, and Mac OSX universal operating systems.

Tools > Add-ons > QPL6

Then, start the QPL interface programs by first launching Komodo and then
opening QPL6 in the tools menu.

Tools > QPL6...

or use the buttons on the QPL toolbar (look for the red "Q").
